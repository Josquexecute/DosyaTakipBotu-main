# Üretim Adayı Kontrol Notları

Bu dosya adı geriye dönük audit uyumluluğu için korunur. İçerik HasarBotu v0.6.0 üretim adayı kabul sürecini anlatır.

## Üretim Adayı Kabul Kapısı

Bir sürüm ofise dağıtılmadan önce:

```bash
npm run typecheck
npm run build
npm run ci
npm run test:behavior
npm run final-office-audit
npm audit
```

komutları temiz geçmelidir.

## Release Aday Kontrolü

```bash
npm run dist:win
npm run release:hash
npm run release:notes
npm run release:dry-run
npm run release:candidate-check
```

Kontrol edilen başlıklar:

- package sürümü ve `APP_VERSION` uyumu,
- kurulum ve taşınabilir EXE varlığı,
- SHA-256 hash çıktıları,
- release notu,
- ofis hedef sürüm marker kaydı,
- fresh build kanıtı.
